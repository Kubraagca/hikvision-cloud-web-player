Network Interface

Table 1 PoE Port Configuration

Function

Description

Get PoE Parameters

Call NET_DVR_GetDVRConfig with NET_DVR_GET_POE_CFG (command No.: 6114). And the PoE parameters are returned by the lpOutBuffer in the structure NET_DVR_POE_CFG.

Set PoE Parameters

Call NET_DVR_SetDVRConfig with NET_DVR_SET_POE_CFG (command No.: 6115), and set the input parameter pointer lpInBuffer to the structure NET_DVR_POE_CFG.
Table 2 UPnP Port Configuration

Function

Description

Get UPnP Parameters

Call NET_DVR_GetDVRConfig with NET_DVR_GET_NAT_CFG (command No.: 6111). And the UPnP parameters are returned by the output buffer pointer (lpOutBuffer) in the structure NET_DVR_NAT_CFG.

Set UPnP Parameters

Call NET_DVR_SetDVRConfig with NET_DVR_SET_NAT_CFG (command No.: 6112) and set the input buffer pointer (lpInBuffer) to the structure NET_DVR_NAT_CFG.

Get UPnP Port Mapping Status

Call NET_DVR_GetUpnpNatState. And the mapping status is returned by the parameter lpState in the structure NET_DVR_UPNP_NAT_STATE.
