Device Calibration

Table 1 Accelerometer Zero Bias Calibration

Function

API

Get device capability to check whether device supports accelerometer zero bias calibration

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/capabilities with GET method.

The device capability is returned in the message XML_DeviceCap by lpOutputParam.

If supports, the node <isSupportZeroBiasCalibration> is returned in the message XML_DeviceCap and its value is true.

Calibrate accelerometer zero bias

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/zeroBiasCalibration/channels/<ID>?format=json by PUT method.
Table 2 Installation Angle Calibration

Function

API

Get device capability to check whether device supports installation angle calibration

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/capabilities with GET method.

The device capability is returned in the message XML_DeviceCap by lpOutputParam.

If supports, the node <isSupportInstallationAngleCalibration> is returned in the message XML_DeviceCap and its value is true.

Get capability of installation angle calibration

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/installationAngleCalibration/channels/<ID>/capabilities?format=json with GET method.

Get installation angle calibration status or calibrate installation angle

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/installationAngleCalibration/channels/<ID>?format=json with GET or PUT method.

Table 3 Laser Optical Axis

Function

API

Get configuration capability of laser optical axis

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/laserOpticalAxis/capabilities?format=json with GET method.

The device capability is returned in the message JSON_LaserOpticalAxisCap by lpOutputParam.

Get laser optical axis parameters

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/laserOpticalAxis?format=json with GET method.

The parameters are returned in the message JSON_LaserOpticalAxis by lpOutputParam.

Set laser optical axis parameters

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/laserOpticalAxis?format=json with GET or PUT methd and set lpInputParam to the message JSON_LaserOpticalAxis.

Adjust laser optical axis

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/laserOpticalAxis/goto?format=json with PUT method and set lpInputParam to the message JSON_LaserOpticalAxis_direction.
