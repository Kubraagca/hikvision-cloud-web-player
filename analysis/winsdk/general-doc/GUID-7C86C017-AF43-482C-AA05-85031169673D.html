System Requirements

To integrate functions by HCNetSDK, both the Windows and Linux operating systems are available. The followings show the requirement details.

For Windows Operating System with 32-bit

Windows8/Windows7/Vista/XP/2000, Windows Server 2012/2008/2003

For Windows Operating System with 64-bit

Windows10/Windows8/Windows7, Windows Server 2012/2008

For Linux Operating System with 32-bit

GCC Version 4.1.2 or Above

OS: CentOS 5.4, Redhat AS5.4, Ubuntu 9.10, Fedora12 or Above

For Linux Operating System with 64-bit

GCC Version 4.1.2 or Above

OS: CentOS 5.4, Redhat AS5.4, Ubuntu 9.10, SUSE10 or Above
