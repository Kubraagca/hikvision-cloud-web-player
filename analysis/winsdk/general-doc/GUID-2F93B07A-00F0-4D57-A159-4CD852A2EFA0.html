Two-Way Audio

The two-way audio application provides the functions of voice talk between control center and device, audio forwarding, encoding and decoding audio, and audio broadcast.

Note:

  • When developing under Windows (64-bit) or Linux system, only the audio forwarding application is supported.

  • When developing under Windows 7 system, the APIs used to implement two-way audio and audio broadcast application will return "FALSE" if there is no external audio device.
