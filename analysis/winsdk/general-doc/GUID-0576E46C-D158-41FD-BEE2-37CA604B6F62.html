Upload Files

You can upload files with different types, i.e., recording files, configuration files, pictures, and so on, of device to the cloud storage for some typical uses.

  • Make sure you have called NET_DVR_Init to initialize the development environment.

  • Make sure you have called NET_DVR_Login_V40 to log in to the device.

Figure 1 Programming Flow of Uploading Files

  1. Call NET_DVR_UploadFile_V40 to start uploading file.
  2. Call NET_DVR_GetUploadState to get the uploading progress.
    Note:

    If the progress is "4" (network disconnected), you should stop uploading first and perform step1 again when the network restored.

  3. Call NET_DVR_UploadClose to stop uploading file.

Call NET_DVR_Logout and NET_DVR_Cleanup to log out and release resources.
