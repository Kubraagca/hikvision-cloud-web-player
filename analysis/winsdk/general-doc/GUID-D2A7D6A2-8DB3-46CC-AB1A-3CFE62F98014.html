Video/Image Settings

Camera Video/Image Settings

Basic Parameters

Get camera image parameters

Call NET_DVR_GetDVRConfig with "NET_DVR_GET_CCDPARAMCFG" (command No.: 1067) and set lChannel to "0XFFFFFFFF".

The camera image parameters are returned in the structure NET_DVR_CAMERAPARAMCFG by lpOutBuffer.

Set camera image parameters

Call NET_DVR_SetDVRConfig with "NET_DVR_SET_CCDPARAMCFG" (command No: 1068), set lChannel to "0XFFFFFFFF", and set lpInBuffer to the structure NET_DVR_CAMERAPARAMCFG.

Correct defective pixel

Call NET_DVR_RemoteControl with "NET_DVR_DPC_CTRL" (command No.: 3410) and set lpInBuffer to the structure NET_DVR_DPC_PARAM.

Note:

This function is only available for certain camera models.

ISP Parameters

Get ISP (Image Signal Processing) parameters

Call NET_DVR_GetDVRConfig with "NET_DVR_GET_ISP_CAMERAPARAMCFG" (command No.: 3255).

The ISP parameters are returned in the structure NET_DVR_ISP_CAMERAPARAMCFG bylpOutBuffer.

Set ISP (Image Signal Processing) parameters

Call NET_DVR_SetDVRConfig with "NET_DVR_SET_ISP_CAMERAPARAMCFG" (command No.: 3256) and set lpInBuffer to the structure NET_DVR_ISP_CAMERAPARAMCFG.

Shielded Area Parameters

Get shielded area parameters of all channels

Call NET_DVR_GetDVRConfig with "NET_DVR_GET_VCA_MASK_REGION" (command No.: 167).

The parameters are returned by lpOutBuffer in the structure NET_VCA_MASK_REGION_LIST.

Get shielded area parameters of a channel by event

Call NET_DVR_STDXMLConfig to transmit /ISAPI/Intelligent/channels/<ID>/Shield/EventType by GET method.

The parameters are returned by lpOutBuffer in the message XML_Shield.

Set shielded area parameter of a channel by event

Call NET_DVR_STDXMLConfig to transmit /ISAPI/Intelligent/channels/<ID>/Shield/EventType by PUT method and set lpInBuffer to the message XML_Shield.

On Screen Display (OSD) Settings

Get OSD capability

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/Video/capabilities by GET method.

The capability is returned in the message XML_VideoCap by lpOutputParam.

OSD Language

Get text language parameters of OSD

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/Video/inputs/OSDLanguage by GET method.

The OSD language parameters are returned in the message XML_Language by lpOutputParam.

Set text language parameters of OSD

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/Video/inputs/OSDLanguage by PUT method and set lpInputParam to the message XML_Language.

Intelligent Information Display

Display intelligent information on live video

Call NET_DVR_RenderPrivateData and set information type (iIntelType) as needed.

Display intelligent thermal information on live video

Call PlayM4_RenderPrivateDataEx and set iIntelType to "RENDER_FIRE_DETCET" or "RENDER_TEM".

Smart Event Rule Display

Get capability of displaying rule information of smart events on video

Call NET_DVR_STDXMLConfig to transmit /ISAPI/Streaming/channels/<ID>/smartOverlap/capabilities?format=json by GET method.

The capability is returned in the message JSON_SmartOverlapCap by lpOutputParam.

Note:

To check whether the function is supported by the device, you can call NET_DVR_STDXMLConfig to transmit /ISAPI/System/capabilities by GET method to get device capability XML_DeviceCap. If it supports, the node <supportSmartOverlapChannles> will be returned and its value is "true".

Get parameters of displaying rule information of smart events on video

Call NET_DVR_STDXMLConfig to transmit /ISAPI/Streaming/channels/<ID>/smartOverlap?format=json by GET method.

The parameters are returned in the message JSON_SmartOverlap by the output parameter lpOutputParam.

Set parameters of displaying rule information of smart events on video

Call NET_DVR_STDXMLConfig to transmit /ISAPI/Streaming/channels/<ID>/smartOverlap?format=json by PUT method and set lpInputParam to the message JSON_SmartOverlap.

Temperature Range Display

Get configuration capability of displayed temperature range

Call NET_DVR_STDXMLConfig to transmit /ISAPI/Image/channels/<ID>/tempRange/capabilities by GET method.

The capability is returned in the message XML_Cap_tempRange by lpOutputParam.

Get displayed temperature range

Call NET_DVR_STDXMLConfig to transmit /ISAPI/Image/channels/<ID>/tempRange by GET.

The parameters are returned in the message XML_tempRange by lpOutputParam.

Set displayed temperature range

Call NET_DVR_STDXMLConfig to transmit /ISAPI/Image/channels/<ID>/tempRange by PUT method and set lpInputParam to the message XML_tempRange.
