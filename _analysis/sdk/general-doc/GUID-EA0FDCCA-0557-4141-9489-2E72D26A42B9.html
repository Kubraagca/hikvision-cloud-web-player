Peripherals Settings

Peripherals Configuration Capability

Peripherals Configuration Capability

Call NET_DVR_GetSTDAbility, set dwAbilityType to "NET_DVR_GET_EXTERNALDEVICE_CAPABILITIES" (value: 3722) and set lpCondBuffer of structure NET_DVR_STD_ABILITY to a 4-byte channel No. for getting peripherals configuration capability.

The capability is returned in the message XML_Cap_ExternalDevice

Built-in Supplement Light

Configuration Capability

Call NET_DVR_GetSTDAbility, set dwAbilityType to "NET_DVR_GET_SUPPLEMENTLIGHT_CAPABILITIES" (value: 3727) and set lpCondBuffer of structure NET_DVR_STD_ABILITY to a 4-byte channel No. for getting configuration capability of built-in supplement light.

The configuration capability is returned in the message XML_Cap_SupplementLight by lpOutBuffer of structure NET_DVR_STD_ABILITY.

Configuration Parameters

Call NET_DVR_GetSTDConfig with "NET_DVR_GET_SUPPLEMENTLIGHT" (command No.: 3728) and set lpCondBuffer of structure NET_DVR_STD_CONFIG to a 4-byte supplement No. for getting parameters of built-in supplement light.

The configuration parameters are returned in the structure NET_DVR_BUILTIN_SUPPLEMENTLIGHT by lpOutBuffer of structure NET_DVR_STD_CONFIG.

Call NET_DVR_SetSTDConfig with "NET_DVR_SET_SUPPLEMENTLIGHT" (command No.: 3729), setlpCondBuffer and lpInBuffer of structure NET_DVR_STD_CONFIG to a 4-byte supplement No. and NET_DVR_BUILTIN_SUPPLEMENTLIGHT, respectively, for setting parameters of built-in supplement light.

POS (Point of Sale)

POS Capability

Call NET_DVR_GetDeviceAbility, set dwAbilityType to "DEVICE_ABILITY_INFO" (value: 0x011), and set pInBuf to the message XML_Desc_POSAbility for getting POS capability.

The POS capability is returned in the message XML_POSAbility by pOutBuf.

Filter Rule of POS Information

Call NET_DVR_GetDVRConfig with "NET_DVR_GET_POS_FILTER_CFG" (command No.: 6148) and set lChannel to the rule ID (which starts from 1) for getting filter rule of POS information.

The filter rule parameters are returned in the structure NET_DVR_POS_FILTER_CFG by lpOutBuffer.

Call NET_DVR_SetDVRConfig with "NET_DVR_SET_POS_FILTER_CFG" (command No.: 6149), set lChannel to the rule ID (which starts from 1), and set lpInBuffer to NET_DVR_POS_FILTER_CFG for setting filter rule of POS information.

Connection Mode of DVR and POS

Call NET_DVR_GetDVRConfig with "NET_DVR_GET_CONNECT_POS_CFG" (command No.: 6150) and set lChannel to the rule ID (which starts from 1) for getting connection mode of DVR and POS.

The connection mode parameters are returned in the structure NET_DVR_CONNECT_POS_CFG by lpOutBuffer.

Call NET_DVR_SetDVRConfig with "NET_DVR_SET_CONNECT_POS_CFG" (command No.: 6151), set lChannel to the rule ID (which starts from 1), and set lpInBuffer to NET_DVR_CONNECT_POS_CFG for setting connection mode of DVR and POS.

Linkage between Filter Rule and Channel

Call NET_DVR_GetDeviceConfig with "NET_DVR_GET_CHAN_FILTER_CFG" (command No.: 6152), set dwCount to 1, and set lpInBuffer to the structure NET_DVR_CHANNEL_GROUP for getting linkage information between filter rule and channel.

The linkage information is returned in the structure NET_DVR_CHAN_FILTER_CFG by lpOutBuffer.

Call NET_DVR_SetDeviceConfig with "NET_DVR_SET_CHAN_FILTER_CFG" (command No.: 6153), set dwCount to 1, set lpInBuffer to the structure NET_DVR_CHANNEL_GROUP, and set lpInParamBuffer to the structure NET_DVR_CHAN_FILTER_CFG for setting the linkage between filter rule and channel.

Hidden Configuration of POS Information

Call NET_DVR_GetSTDAbility and set dwAbilityType to "NET_DVR_GET_HIDDEN_INFORMATION_CAPABILITIES" (value: 3608) for getting hidden configuration capability of POS information.

The configuration capability is returned in the message XML_Cap_HiddenInformation by lpOutBuffer of structure NET_DVR_STD_ABILITY.

Call NET_DVR_GetSTDConfig with "NET_DVR_GET_HIDDEN_INFORMATION" (command No.: 6721) to get hidden parameters of POS information.

The hidden parameters are returned in the structure NET_DVR_HIDDEN_INFORMATION_CFG by lpOutBuffer of structure NET_DVR_STD_CONFIG.

Call NET_DVR_SetSTDConfig with "NET_DVR_SET_HIDDEN_INFORMATION" (command No.: 6722), set lpInBuffer of structure NET_DVR_STD_CONFIG to NET_DVR_HIDDEN_INFORMATION_CFG for setting hidden parameters of POS information.
