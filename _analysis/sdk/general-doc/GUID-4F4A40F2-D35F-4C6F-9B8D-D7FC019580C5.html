Multicast

The multicast is to realize the group communication where information is addressed to a group of destination computers simultaneously.

Get Parameters of Multicast Based on NPQ

Call NET_DVR_STDXMLConfig to pass through the request URL: GET /ISAPI/System/Network/extension.

The parameters are returned in the message XML_networkExtension by the output parameter (lpOutputParam).

Set Parameters of Multicast Based on NPQ

Call NET_DVR_STDXMLConfig to pass through the request URL: PUT /ISAPI/System/Network/extension and set the input parameter (lpInputParam) to the message XML_networkExtension.
