HDD Management

The HDD (Hard Disk Drive) is used for storage. Before storing information in HDD, you must format it, and you can also reallocate its space for better storage management.

HDD Configuration

Get HDD management and configuration capability

Call NET_DVR_STDXMLConfig to transmit /ISAPI/ContentMgmt/Storage/hdd/capabilities by GET method.

The capability is returned in the message XML_Cap_hddList by lpOutputParam.

Get HDD parameters

Call NET_DVR_GetDVRConfig with "NET_DVR_GET_HDCFG_V50" (command No.: 4153) and set lChannel to the HDD group No. (starts from 0 and up to 33 HDDs are allowed in a group).

The HDD parameters are returned in the structure NET_DVR_HDCFG_V50 by the output buffer (lpOutBuffer)

Set HDD parameters

Call NET_DVR_SetDVRConfig with "NET_DVR_SET_HDCFG_V50" (command No.: 4154), set lChannel to the HDD group No. (starts from 0 and up to 33 HDDs are allowed in a group), and set lpInBuffer to NET_DVR_HDCFG_V50.

Get parameters of a HDD

Call NET_DVR_STDXMLConfig to transmit /ISAPI/ContentMgmt/Storage/hdd/<ID> by GET method.

The parameters are returned in the message XML_hdd by lpOutputParam.

Set parameters of a HDD

Call NET_DVR_STDXMLConfig to transmit /ISAPI/ContentMgmt/Storage/hdd/<ID> by PUT method and set lpInputParam to XML_hdd.

Verify HDD encryption password

Call NET_DVR_STDXMLConfig to transmit /ISAPI/ContentMgmt/Storage/hdd/<ID>/encryptVerfy?format=json by PUT method and set lpInputParam to JSON_EncryptVerfy.

HDD Formatting

Format a HDD

  1. Call NET_DVR_FormatDisk to start formatting a HDD remotely.

  2. Call NET_DVR_GetFormatProgress to get the formatting progress of the HDD.

  3. Call NET_DVR_CloseFormatHandle to stop formatting the HDD.

Format HDDs in a batch

Call NET_DVR_STDXMLConfig to transmit /ISAPI/ContentMgmt/Storage/hdd/specifyHddFormat?format=json by PUT method and set lpInputParam to JSON_HddFormatList.

Format an encrypted HDD

Call NET_DVR_STDXMLConfig to transmit /ISAPI/ContentMgmt/Storage/hdd/<ID>/encryptFormat?format=json by PUT method and set lpInputParam to JSON_EncryptFormat.
