en-US

/ISAPI/Event/schedules/vibrationDetection/<ID>

Get or set arming schedule of vibration detection by channel.

Request URI Definition

Table 1 GET /ISAPI/Event/schedules/vibrationDetection/<ID>

Method

GET

Description

Get arming schedule of vibration detection by channel.

Query

None.

Request

None.

Response

Succeeded: XML_Schedule

Failed: XML_ResponseStatus
Table 2 PUT /ISAPI/Event/schedules/vibrationDetection/<ID>

Method

PUT

Description

Set arming schedule of vibration detection by channel.

Query

None.

Request

XML_Schedule

Response

XML_ResponseStatus

Remarks

The <ID> in the request URI should be set in the format of "vibrationDetection-<channelID>", e.g., /ISAPI/Event/schedules/vibrationDetection/vibrationDetection-101.
