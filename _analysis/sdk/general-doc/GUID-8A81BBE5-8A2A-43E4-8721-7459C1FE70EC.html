Wireless Network

Wireless Hotspot

Get Wireless Hotspot Capability

Call NET_DVR_GetSTDAbility with "NET_DVR_GET_WIRELESSSERVER_CAPABILITIES" (command No.: 3716) and set lpCondBuffer to a 4-byte wireless NIC No.

The capability is returned in the message XML_Cap_WirelessServer by lpOutBuffer.

Get Wireless Hotspot Parameters

Call NET_DVR_GetSTDConfig with "NET_DVR_GET_WIRELESSSERVER" (command No.: 3717) and set lpCondBuffer to a 4-byte wireless NIC No.

The parameters are returned in the structure NET_DVR_WIRELESSSERVER by lpOutBuffer.

Set Wireless Hotspot Parameters

Call NET_DVR_SetSTDConfig with "NET_DVR_SET_WIRELESSSERVER" (command No.: 3718), set lpCondBuffer to a 4-byte wireless NIC No., and specify lpInBuffer as the structure NET_DVR_WIRELESSSERVER.

Wireless Dial-up

Get Wireless Dial-up Configuration Capability

  • Option 1: Call NET_DVR_GetSTDAbility with "NET_DVR_GET_WIRELESSDIAL_CAPABILITIES" (command No.: 3580) and set lpCondBuffer to a 4-byte wireless NIC No.

  • Option 2: Call NET_DVR_STDXMLConfig to transmit GET /ISAPI/System/Network/WirelessDial/Interfaces/<ID>/dial/capabilities.

The capability is returned in the message XML_Cap_Dial by lpOutBuffer or lpOutputParam.

Get Wireless Dial-up Parameters

  • Option 1: Call NET_DVR_GetSTDConfig with "NET_DVR_GET_WIRELESS_DIAL" (command No.: 3578) and set lpCondBuffer to a 4-byte wireless NIC No.

    The parameters are returned in the structure NET_DVR_WIRELESSDIAL_CFG by lpOutBuffer.

  • Option 2: Call NET_DVR_STDXMLConfig to transmit GET /ISAPI/System/Network/WirelessDial/Interfaces/<ID>/dial.

    The parameters are returned in the message XML_Dial by lpOutputParam.

Set Wireless Dial-up Parameters

  • Option 1: Call NET_DVR_SetSTDConfig with "NET_DVR_SET_WIRELESS_DIAL" (command No.: 3579), set lpCondBuffer to a 4-byte wireless NIC No., and set the input parameter lpInputParam to the structure NET_DVR_WIRELESSDIAL_CFG.

  • Option 2: Call NET_DVR_STDXMLConfig to transmit PUT /ISAPI/System/Network/WirelessDial/Interfaces/<ID>/dial and set lpInputParam to the message XML_Dial.

Get Wireless Dial-up Schedule Capability

Call NET_DVR_GetSTDAbility with "NET_DVR_GET_WIRELESSDIAL_SCHEDULE_CAPABILITIES" (command No.: 3592) and set lpCondBuffer to a 4-byte wireless NIC No.

The capability is returned in the message XML_Cap_Schedule by lpOutBuffer.

Get Wireless Dial-up Schedule Parameters

  • Option 1: Call NET_DVR_GetSTDConfig with "NET_DVR_GET_WIRELESSDIAL_SCHEDULE" (command No.: 3581) and set lpCondBuffer to a 4-byte wireless NIC No.

    The parameters are returned in the structure NET_DVR_WIRELESSDIAL_SCHEDULE by lpOutBuffer.

  • Option 2: Call NET_DVR_STDXMLConfig to transmit GET /ISAPI/System/Network/WirelessDial/Interfaces/<ID>/schedule.

    The parameters are returned in the message XML_Schedule by lpOutputParam.

Set Wireless Dial-up Schedule Parameters

  • Call NET_DVR_SetSTDConfig with "NET_DVR_SET_WIRELESSDIAL_SCHEDULE" (command No.: 3582), set lpCondBuffer to a 4-byte wireless NIC No., and set lpInputParam to the structure NET_DVR_WIRELESSDIAL_SCHEDULE.

  • Option 2: Call NET_DVR_STDXMLConfig to transmit PUT /ISAPI/System/Network/WirelessDial/Interfaces/<ID>/schedule and set lpInputParam to the message XML_Schedule.

Get Wireless Dial-up Status Parameters

Network Keepalive

Get Network Keepalive Capability

Call NET_DVR_STDXMLConfig to transmit the request URL: GET /ISAPI/System/Network/WirelessDial/Interfaces/<ID>/keepAlive/capabilities?format=json.

The capability is returned in the message JSON_KeepAliveCap by the output parameter lpOutputParam.

Get Network Keepalive Parameters

Call NET_DVR_STDXMLConfig to transmit the request URL: GET /ISAPI/System/Network/WirelessDial/Interfaces/<ID>/keepAlive?format=json.

The parameters are returned in the message JSON_KeepAlive by the output parameter lpOutputParam.

Set Network Keepalive Parameters

Call NET_DVR_STDXMLConfig to transmit the request URL: PUT /ISAPI/System/Network/WirelessDial/Interfaces/<ID>/keepAlive?format=json and set the input parameter lpInputParam to the message JSON_KeepAlive.

Test Network Keepalive

Call NET_DVR_STDXMLConfig to transmit the request URL: POST /ISAPI/System/Network/WirelessDial/Interfaces/<ID>/keepAlive/test?format=json and set the input parameter lpInputParam to the message JSON_KeepAlive.
