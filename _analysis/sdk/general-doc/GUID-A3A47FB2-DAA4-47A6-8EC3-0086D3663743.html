Basic Settings

Table 1 Device Basic Parameters

Function

API

Get device basic parameters

Call NET_DVR_GetDVRConfig with "NET_DVR_GET_DEVICECFG_V50" (command No.: 3801).

The device basic parameters are returned in the structure NET_DVR_DEVICECFG_V50 by lpOutBuffer.

Set device basic parameters

Call NET_DVR_SetDVRConfig with "NET_DVR_SET_DEVICECFG_V50" (command No.: 3802) and set lpInBuffer to the structure NET_DVR_DEVICECFG_V50.
Table 2 Device Language

Function

API

Get device language configuration capability

Call NET_DVR_STDXMLConfig to transmit/ISAPI/System/DeviceLanguage/capabilities with GET method.

The capability is returned in the message XML_Cap_DeviceLanguage by the output parameter lpOutputParam.

Get device language

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/DeviceLanguage with GET method.

Set device language

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/DeviceLanguage with PUT method and set lpInputParam to the message XML_DeviceLanguage.
Table 3 Configuration Files

Function

API

Securely import configuration files to device

Call NET_DVR_UploadFile_V40, set the file type dwUploadType to "UPLOAD_SECURITY_CFG_FILE", and set lpInBuffer to the structure NET_DVR_SECURITY_CFG_FILE_COND.

The device basic parameters are returned in the structure NET_DVR_DEVICECFG_V50 by lpOutBuffer.

Note:

You can check whether the device supports importing calibration file by calling API NET_DVR_STDXMLConfig to transmit /ISAPI/System/capabilities with GET method. When the node isSupportCalibrationFile exists and its value is true in the returned device capability (XML_DeviceCap), it indicates the function is supported.

Get configuration files importing progress

Call NET_DVR_GetUploadState.

Stop importing configuration files

Call NET_DVR_UploadClose.
Table 4 Camera Software Service

Function

API

Get configuration capability of software service

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/Software/channels/<ID>/capabilities with GET method.

The configuration capability is returned in the message XML_Cap_SoftwareService by IpOutputParam.

Get software service parameter

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/Software/channels/<ID> with GET method.

The configuration parameter is returned in the message XML_SoftwareService by IpOutputParam.

Set software service parameter

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/Software/channels/<ID> with PUT method and set IpInputParam to the message XML_SoftwareService.
Table 5 Device Attribute Code

Function

API

Get device attribute code

Call NET_DVR_STDXMLConfig to transmit /ISAPI/System/deviceInfo/characteristicCode?format=json with GET method.

The parameters are returned in the message JSON_Characteristic by IpOutputParam.
