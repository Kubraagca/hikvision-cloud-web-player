Other File Operations

Lock or Unlock Video Files

Get video file locking and unlocking capability

Call NET_DVR_STDXMLConfig to transmit /ISAPI/ContentMgmt/record/control/locks/capabilities by GET method.

The capability is returned in the message XML_LockCap by lpOutputParam.

Lock or unlock video files by name

Call NET_DVR_STDXMLConfig to transmit /ISAPI/ContentMgmt/record/control/locks/name by PUT method and set lpInputParam to XML_LockByName.

Add Remarks to Video File

Get capability of adding remarks to video files

Call NET_DVR_STDXMLConfig to transmit /ISAPI/ContentMgmt/record/control/remark/capabilities?format=json by GET method.

The capability is returned in the message JSON_Cap_RemarkList by lpOutputParam.

Add remarks to video files

Call NET_DVR_STDXMLConfig to transmit /ISAPI/ContentMgmt/record/control/remark?format=json by PUT method and set lpInputParam to JSON_RemarkList.
Note:

To check whether the device supports adding remarks to the video files, you can call NET_DVR_STDXMLConfig to transmit /ISAPI/ContentMgmt/capabilities by GET method to get the device storage capability (XML_RacmCap). If it supports, the node <isSupportRemark> will be returned and its value is "true".
